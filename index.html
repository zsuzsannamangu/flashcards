Javascript and jQuery Flashcards

  • JavaScript (JS)

    JavaScript is a programming language used to make web pages interactive.

  • Operators

    Assignment operator: Changes the value of the variable on the left of the operator.

    Comparison operator: Does not change any values, but returns a boolean (true or false) depending on whether the statement evaluates as true or false.

  • Variables

    Variables can be thought of as containers used to store information. They allow for a way to label data with a descriptive name. Variables, in JavaScript, are written in lower camelcase. Variables are set using the var keyword. Variables can be used in arithmetic.

  • Functions

    A function is a block of code that performs an action and returns a result; optionally takes arguments.

  • Methods

    A method is an action run on a piece of data; you can think of it as a message you send to a piece of data, and the result is the response.

  • Arguments

    An argument is what you pass into a function or method. Some methods take one or more arguments that provide the method with additional information to help it perform its action.

  • Parameters

    A parameter is a variable that's assigned to the argument.

  • Return

    The return value is the method's response.

  • Chaining methods

    Calling a method directly on the return value of another method.

  • Strings

    A JavaScript data type that represents the exact text of whatever is enclosed in the quotes. Strings are surrounded with quotation marks.
